Dans la série des Ateliers Back To Basics Silex Labs vous invite dans le "Le Dojo Javascript" un atelier en mode contributhon pour coder ensemble un jeu en Javascript !!

Programme

Entrez dans le “dojo” Javascript

Silex Labs vous  propose un atelier en mode “dojo” pour “Javascripter” ensemble !

Une session de dojo s’organise de manière assez simple : une machine, un pilote qui est aux commandes pour 5 minutes, un copilote qui catch les erreurs, un public attentif ! Une fois le temps écoulé, le pilote rejoint le public, le copilote est promu aux commandes et choisit un membre de l'audience (au hasard ?) pour l’assister.

Ainsi, tout le monde a sa chance, c’est le moment de montrer vos coding tips & skills ou de vous inspirer de celles des autres.

 Objectif

Ce premier dojo sera consacré à développer un petit jeu simple en Javascript.
