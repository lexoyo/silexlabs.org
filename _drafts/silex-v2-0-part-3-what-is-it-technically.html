This article is part of a series of articles about the new version of Silex HTML5 editor.
Designers will find out what they can do with it. Web developers will have an insight of the technologies behind Silex v2.

As stated in the first article about Silex v2 and in the the second article about Silex use cases, there is room for improvements, and several important functionalities are still missing, but Silex is ready for the first contact with designers.

Silex categorization

CMS, concrete5, wordpress

same
open source, free as in free beer and free speach
extensible

different
design first
100% javascript, client side and server side
produce simple HTML files

Website builders, Wix, Jimdo, Weebly...

same
client side
targets designers

different
produce simple HTML files, which is standard
open source, free as in free beer and free speach
you must handle hosting
- we had for Silex v1 and we will try to set it up for Silex v2, a partnership with a hosting company to provide professional hosting automatically plugged in with Silex - then the experience would be the same as with Wix or Jimdo etc
- you can host your files on any hosting plan, even free ones - which is better than with free wix hosting because you will not have any limitations, but you have to handle the transfer of your files to the host

Static site generators, Jekyll, Middleman...

same
static files for better security, performance, cheap hosting

different
wysiwyg / drag'n drop
for web designers and not only developers
cloud based, no install
